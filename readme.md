# Patterns Portfolio

> **Patterns Portfolio** is a clean and modern block-based WordPress theme designed for designers, photographers, artists, and creative professionals.

## Description

Patterns Portfolio is a sleek and professional Full Site Editing (FSE) WordPress theme, perfect for building a striking online presence for designers, photographers, artists, and creative professionals. Designed to showcase portfolios, case studies, and client testimonials, this block-based theme empowers you to create a polished and personalized website. With its FSE capabilities, Patterns Portfolio enables seamless customization of headers, footers, layouts, and global styles directly within the WordPress Site Editor.

## Features

- **Full Site Editing (FSE):** Customize every aspect of your website using the WordPress block editor.
- **Elegant Design:** Perfect for designers, photographers, artists, and creative professionals.
- **Responsive Layouts:** Optimized for seamless display on all devices.
- **Pre-designed Templates:** Includes layouts for showcasing feature sections, services, about, appointments, prices, portfolios, team, testimonials, contact page, and more.
- **Custom Block Patterns:** Simplify website creation with ready-to-use patterns.
- **SEO Optimized:** Built with best practices for improved search engine rankings.
- **High Performance:** Lightweight and optimized for fast loading.

## Getting Started

### Installation

- Clone the repository to your `/wp-content/themes/` directory:

  ```bash
  git clone https://github.com/patternswp/patterns-portfolio.git
  ```

- Or download and upload the theme files to `/wp-content/themes/patterns-portfolio`.

### Activate the Theme

Activate the theme via the WordPress admin dashboard under **Appearance > Themes**.

### Start Customization

With Full Site Editing support, you can customize the theme through the WordPress Site Editor by navigating to **Appearance > Editor**. Modify headers, footers, layouts, and global styles to suit your portfolio needs.
